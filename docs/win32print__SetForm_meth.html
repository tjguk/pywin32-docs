win32print.SetForm

SetForm(hprinter, FormName, Form)

Change information for a form

Parameters

hprinter : PyPrinterHANDLE

Printer handle as returned by win32print::OpenPrinter

FormName : PyUnicode

Name of form

Form : dict

FORM_INFO_1 dictionary

Return Value

Returns None on success
