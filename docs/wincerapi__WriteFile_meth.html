wincerapi.WriteFile

int, int = WriteFile(hFile, data )

Writes a string to a file

Parameters

hFile : PyHANDLE/int

Handle to the file

data : string

The data to write.

Return Value

The result is a tuple of (errCode, nBytesWritten). errCode will always be zero (until overlapped IO is supported!)
