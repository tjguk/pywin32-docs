pywintypes.IID

PyIID = IID(iidString, is_bytes )

Creates a new IID object

Parameters

iidString : string/Unicode

A string representation of an IID, or a ProgID.

is_bytes=False : bool

Indicates if the first param is actually the bytes of an IID structure.
