win32security.GetBinarySid

PySID = GetBinarySid(SID)

Accepts a SID string (eg: S-1-5-32-544) and returns the SID as a PySID object.

Parameters

SID : string

Textual representation of a SID. Textual SID example: S-1-5-32-544
