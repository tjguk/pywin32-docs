win32security.OpenThreadToken

PyHandle = OpenThreadToken(handle, desiredAccess , openAsSelf )

Opens the access token associated with a thread.

Parameters

handle : PyHANDLE

handle to thread

desiredAccess : int

access to process

openAsSelf : int

Flag for process or thread security
