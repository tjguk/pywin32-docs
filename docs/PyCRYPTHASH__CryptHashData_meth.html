PyCRYPTHASH.CryptHashData

CryptHashData(Data, Flags)

Adds data to the hash

Parameters

Data : string

Data to be hashed

Flags=0 : int

CRYPT_USERDATA or 0

Comments

If Flags is CRYPT_USERDATA, provider is expected to prompt user to enter data. MSDN says that MS CSPs ignore this flag
