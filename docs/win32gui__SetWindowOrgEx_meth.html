win32gui.SetWindowOrgEx

(int,int) = SetWindowOrgEx(hdc, X , Y )

Changes the window origin for a DC

Parameters

hdc : PyHANDLE

Handle to a device context

X : int

New X coord in logical units

Y : int

New Y coord in logical units

Return Value

Returns the previous origin
