win32gui.OffsetRgn

int = OffsetRgn(hrgn, XOffset , YOffset )

Relocates a region

Parameters

hrgn : PyGdiHANDLE

Handle to a region

XOffset : int

Horizontal offset

YOffset : int

Vertical offset

Return Value

Returns type of region (COMPLEXREGION, NULLREGION, or SIMPLEREGION)
