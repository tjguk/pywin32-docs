shell.StringAsCIDA

PyIDL, list = StringAsCIDA(pidl)

Given a CIDA as a raw string, return the folder PIDL and list of children

Parameters

pidl : string

The PIDL as a raw string.

Return Value

The result is the PIDL of the folder, and a list of child PIDLs.
