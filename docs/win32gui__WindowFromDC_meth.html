win32gui.WindowFromDC

PyHANDLE = WindowFromDC(hDC)

Finds the window associated with a device context

Parameters

hDC : PyHANDLE

Handle to a device context

Return Value

Returns a handle to the window, or 0 if the DC is not associated with a window
