win32net.NetGroupAdd

NetGroupAdd(server, level, data)

Creates a new group.

Parameters

server : string/PyUnicode

The name of the server, or None.

level : int

The information level contained in the data

data : PyGROUP_INFO_*

A dictionary holding the group data.
