win32gui.FillRect

FillRect(hDC, rc, hbr)

Fills a rectangular area with specified brush

Parameters

hDC : PyHANDLE

Handle to a device context

rc : PyRECT

Rectangle to be filled

hbr : PyGdiHANDLE

Handle to brush to be used to fill area
