win32crypt.CertAlgIdToOID

string = CertAlgIdToOID(AlgId)

Converts an integer ALG_ID to it's szOID_ string representation

Parameters

AlgId : int

An algorithm identifier

Comments

If there is no corresponding OID, None is returned
