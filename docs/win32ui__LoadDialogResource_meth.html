win32ui.LoadDialogResource

list = LoadDialogResource(idRes, dll )

Loads a dialog resource, and returns a list detailing the objects.

Parameters

idRes : int

The ID of the dialog resource to load.

dll=None : PyDLL

The DLL object to load the dialog from.
