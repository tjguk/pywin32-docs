win32gui.CreatePen

PyGdiHANDLE = CreatePen(PenStyle, Width , Color )

Create a GDI pen

Parameters

PenStyle : int

One of win32con.PS_* pen styles

Width : int

Drawing width in logical units. Use zero for single pixel.

Color : int

RGB color value. See win32api::RGB.
