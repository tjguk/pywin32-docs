win32gui.GetDlgItem

HWND = GetDlgItem(hDlg, IDDlgItem )

Retrieves the handle to a control in the specified dialog box.

Parameters

hDlg : PyHANDLE

Handle to a dialog window

IDDlgItem : int

Identifier of one of the dialog's controls
