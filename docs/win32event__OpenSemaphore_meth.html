win32event.OpenSemaphore

PyHANDLE = OpenSemaphore(desiredAccess, bInheritHandle , name )

Returns a handle of an existing named semaphore object.

Parameters

desiredAccess : int

access flag

bInheritHandle : bool

inherit flag

name : PyUnicode

name of semaphore to open.
