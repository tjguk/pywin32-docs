win32file.GetFileTime

(PyTime, PyTime, PyTime) = GetFileTime(handle, creationTime , accessTime , writeTime )

Returns a file's creation, last access, and modification times.

Parameters

handle : PyHANDLE

Handle to the file.

creationTime : PyTime

accessTime : PyTime

writeTime : PyTime

Comments

Times are returned in UTC time.
