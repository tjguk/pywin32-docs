win32event.SetWaitableTimer

SetWaitableTimer(handle, dueTime, period, func, param, resume_state)

Parameters

handle : PyHANDLE

handle to timer

dueTime : long

timer due time

period : int

timer interval

func : object

completion routine - must be None

param : object

completion routine parameter - must be None

resume_state : bool

resume state
