win32gui.SetPixelV

SetPixelV(hdc, X, Y, Color)

Sets the color of a single pixel to an approximation of specified color

Parameters

hdc : PyHANDLE

Handle to a device context

X : int

Horizontal pos

Y : int

Vertical pos

Color : int

RGB color to be set.
