win32evtlog.OpenBackupEventLog

PyEVTLOG_HANDLE = OpenBackupEventLog(serverName, fileName )

Opens a previously saved event log.

Parameters

serverName : PyUnicode

The server name, or None

fileName : PyUnicode

The filename to open
