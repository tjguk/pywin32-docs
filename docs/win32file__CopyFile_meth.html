win32file.CopyFile

CopyFile(from, to, bFailIfExists)

Copies a file

Parameters

from : PyUnicode

The name of the file to copy from

to : PyUnicode

The name of the file to copy to

bFailIfExists : int

Indicates if the operation should fail if the file exists.
