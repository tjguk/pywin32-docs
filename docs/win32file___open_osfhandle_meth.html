win32file._open_osfhandle

int = _open_osfhandle(osfhandle, flags )

Associates a C run-time file handle with a existing operating-system file handle.

Parameters

osfhandle : PyHANDLE

An open file handle

flags : int

O_APPEND,O_RDONLY, or O_TEXT
