win32pipe.popen4

(pipe, pipe) = popen4(cmdstring, mode )

Variation on win32pipe::popen

Parameters

cmdstring : string

The cmdstring to pass to the shell

mode : string

Either 't' or 'b'

Return Value

The result of this function is 2 pipes - the processes stdin, and stdout+stderr combined as a single pipe.
