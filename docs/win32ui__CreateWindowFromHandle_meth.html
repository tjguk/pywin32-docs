win32ui.CreateWindowFromHandle

PyCWnd = CreateWindowFromHandle(hwnd)

Creates a PyCWnd from an integer containing a HWND

Parameters

hwnd : int

The window handle.

Return Value

The result is a PyCWnd (or derived) object, or a win32ui.error exception is raised.
