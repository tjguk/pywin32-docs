win32inet.InternetWriteFile

int = InternetWriteFile(File, Buffer )

Writes data to a handle opened by win32inet::FtpOpenFile.

Parameters

File : PyHINTERNET

Writeable internet handle

Buffer : string

String or buffer containing data to be written
