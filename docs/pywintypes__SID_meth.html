pywintypes.SID

PySID = SID(bufSize)

Creates a new SID object

Parameters

bufSize=32 : int

Size for the SID buffer

Alternative Parameters

buffer

A raw data buffer, assumed to hold the SID data.

Alternative Parameters

idAuthority

The identifier authority.

subAuthorities

A list of sub authorities.
