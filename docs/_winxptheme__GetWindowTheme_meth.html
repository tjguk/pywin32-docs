_winxptheme.GetWindowTheme

PyHTHEME = GetWindowTheme(hwnd)

If window is themed, returns its most recent HTHEME from OpenThemeData() - otherwise, returns NULL.

Parameters

hwnd : int

The window to get the HTHEME of
