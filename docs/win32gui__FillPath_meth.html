win32gui.FillPath

FillPath(hdc)

Fills a path with currently selected brush

Parameters

hdc : PyHANDLE

Handle to a device context that contains a finalized path. See win32gui::EndPath.

Comments

Any open figures are closed and path is deselected from the DC.
