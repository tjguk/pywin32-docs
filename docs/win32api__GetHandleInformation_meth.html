win32api.GetHandleInformation

int = GetHandleInformation(Object)

Retrieves a handle's flags.

Parameters

Object : PyHANDLE

Handle to an object

Comments

Not available on Win98/Me

Return Value

Returns a combination of HANDLE_FLAG_INHERIT, HANDLE_FLAG_PROTECT_FROM_CLOSE
