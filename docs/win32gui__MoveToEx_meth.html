win32gui.MoveToEx

(int, int) = MoveToEx(hdc, X , Y )

Changes the current drawing position

Parameters

hdc : PyHANDLE

Device context handle

X : int

Horizontal pos in logical units

Y : int

Vertical pos in logical units

Return Value

Returns the previous position as (X, Y)
