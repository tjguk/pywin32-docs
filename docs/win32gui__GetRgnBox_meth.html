win32gui.GetRgnBox

int, PyRECT = GetRgnBox(hrgn)

Calculates the bounding box of a region

Parameters

hrgn : PyGdiHANDLE

Handle to a region

Return Value

Returns type of region (COMPLEXREGION, NULLREGION, or SIMPLEREGION) and rectangle in logical units
