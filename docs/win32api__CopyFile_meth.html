win32api.CopyFile

CopyFile(src, dest, bFailOnExist)

Copies an existing file to a new file

Parameters

src : stringPyUnicode

Name of an existing file.

dest : string/PyUnicode

Name of file to copy to.

bFailOnExist=0 : int

Indicates if the operation should fail if the file exists.
