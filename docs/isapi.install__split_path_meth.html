isapi.install.split_path

split_path(path)

Get the parent path and basename.

Parameters

path :

path

>>> split_path('/')

['', '']


>>> split_path('')

['', '']


>>> split_path('foo')

['', 'foo']


>>> split_path('/foo')

['', 'foo']


>>> split_path('/foo/bar')

['/foo', 'bar']


>>> split_path('foo/bar')

['/foo', 'bar']
