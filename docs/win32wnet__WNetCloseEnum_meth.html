win32wnet.WNetCloseEnum

WNetCloseEnum(handle)

Closes a PyHANDLE that represents an Open Enumeration (from win32wnet::WNetOpenEnum)

Parameters

handle : PyHANDLE

The handle to close, as obtained from win32wnet::WNetOpenEnum

Comments

You should perform a WNetClose for each handle returned from win32wnet::WNetOpenEnum.
