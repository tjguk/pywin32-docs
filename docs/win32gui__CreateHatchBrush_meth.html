win32gui.CreateHatchBrush

PyGdiHANDLE = CreateHatchBrush(Style, clrref )

Creates a hatch brush with specified style and color

Parameters

Style : int

Hatch style, one of win32con.HS_* constants

clrref : int

Rgb color value. See win32api::RGB.
