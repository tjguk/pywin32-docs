win32file.GetDriveType

int = GetDriveType(rootPathName)

Determines whether a disk drive is a removable, fixed, CD-ROM, RAM disk, or network drive.

Parameters

rootPathName : string

Return Value

The result is one of the DRIVE_* constants.
