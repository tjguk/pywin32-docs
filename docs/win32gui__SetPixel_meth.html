win32gui.SetPixel

int = SetPixel(hdc, X , Y , Color )

Set the color of a single pixel

Parameters

hdc : PyHANDLE

Handle to a device context

X : int

Horizontal pos

Y : int

Vertical pos

Color : int

RGB color to be set.

Return Value

Returns the RGB color actually set, which may be different from the one passed in
