win32process.GetModuleFileNameEx

PyUNICODE = GetModuleFileNameEx(hProcess, hModule )

Return name of module loaded by another process (uses process handle, not pid)

Parameters

hProcess : PyHANDLE

Process handle as returned by OpenProcess

hModule : PyHANDLE

Module handle
