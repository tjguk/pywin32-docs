win32print.OpenPrinter

PyPrinterHANDLE = OpenPrinter(printer, Defaults )

Retrieves a handle to a printer.

Parameters

printer : string

Printer or print server name. Use None to open local print server.

Defaults=None : dict

PRINTER_DEFAULTS dict, or None
