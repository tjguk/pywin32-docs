win32cred.CredRead

dict = CredRead(TargetName, Type , Flags )

Retrieves a stored credential

Parameters

TargetName : PyUnicode

The target of the credentials to retrieve

Type : int

One of the CRED_TYPE_* constants

Flags=0 : int

Reserved, use 0

Return Value

Returns a PyCREDENTIAL dict
