win32job.UserHandleGrantAccess

UserHandleGrantAccess(hUserHandle, hJob, grant)

Grants or denies access to a handle to a User object to a job that has a user-interface restriction.

Parameters

hUserHandle : PyHANDLE

hJob : PyHANDLE

grant : bool
